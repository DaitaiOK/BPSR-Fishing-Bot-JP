BPSR Fishing Bot（日本語改変版）
導入方法（GitHub・Python初心者向け）

本プロジェクトは Python スクリプトです。
以下の手順に従えば、GitHubやPythonが初めての方でも起動できます。

① 必要なもの

Windows PC

Python 3.8以上

管理者権限で実行できる環境

対応ゲーム（画面解像度：1920×1080 / フルスクリーン）

② Pythonのインストール（未インストールの場合）

以下の公式サイトにアクセス
https://www.python.org/downloads/windows/

「Download Python 3.x.x」をクリック

インストーラー起動時に必ず 「Add Python to PATH」 にチェック

インストール完了後、コマンドプロンプトで以下を入力して確認

python --version

③ プログラムのダウンロード（GitHubから）
方法A（おすすめ・簡単）

このページ右上の 「Code」 ボタンをクリック

「Download ZIP」 を選択

ダウンロードしたZIPを右クリック → すべて展開

方法B（Gitを使える人向け）
git clone https://github.com/DaitaiOK/BPSR-Fishing-Bot-JP.git

④ 必要ライブラリのインストール

展開したフォルダを開く

フォルダ内で Shift + 右クリック → 「ここで PowerShell を開く」

以下を実行

pip install -r requirements.txt


※ エラーが出る場合は

python -m pip install -r requirements.txt

⑤ 起動方法

ゲームを起動

1920×1080 / フルスクリーン に設定

キャラクターを「釣り場でキャスト可能な状態」にする

プロジェクトフォルダ内で PowerShell を 管理者権限で起動

以下を実行

python main.py

⑥ 操作方法（ホットキー）

ゲーム画面を前面にした状態で操作します。

キー	機能
7	ボット起動
8	一時停止 / 再開
9	ROI（画像認識範囲）表示
0	完全停止（終了）

※ 管理者権限での実行が必須です。
※ ROI表示（9キー）はデバッグ用です。

概要

本プロジェクトは、Python + 画像認識（OpenCV）を用いてゲーム内の釣りを自動化するオープンソースのフィッシングボットです。

海外のGPL-3.0ライセンスのオープンソースプロジェクトをベースに、実運用での安定性・実用性を重視して仕様整理・改変を行っています。

動作条件（重要）

Windows環境

Python 3.8以上

ゲームはフルスクリーン

解像度は1920×1080

ゲーム画面を前面表示

キャラクターが「釣り場でキャスト可能な状態」

プログラムは 管理者権限で起動

※ 管理者権限がない場合、マウス・キーボード操作が正常に行われません。
※ 上記条件を満たさない場合、画像認識精度の低下や誤動作が発生する可能性があります。

ステート構成と処理フロー
竿チェック
   ↓
キャスト
   ↓
アタリ待ち
   ↓
ミニゲーム
   ↓
リザルト処理
   ↓
竿チェック（最初に戻る）

各ステートの役割

竿チェック（Checking Rod State）
使用中の釣り竿が正常かを画像認識で確認。破損している場合は自動で交換。

キャスト（Casting Bait State）
キャスト可能な状態を確認後、釣りを開始。

アタリ待ち（Waiting for Bite State）
画面上の変化を監視し、アタリを検知。

ミニゲーム（Playing Minigame State）
魚とのやり取りを自動操作。状況に応じて左右操作を実行。

リザルト処理（Finishing State）
成功・失敗を判定し、次の釣りへ移行。

フェイルセーフ仕様

各ステートにはタイムアウトが設定されており、画像認識に失敗した場合でも一定時間経過後に「竿チェック」ステートへ復帰。
無限停止・フリーズを起こしにくい構成です。

画像認識について

OpenCVのテンプレートマッチングを使用。
テンプレート画像は以下に配置されています：

src/fishbot/assets/templates/


ゲーム側のUI変更で認識しなくなった場合は、該当箇所の画像を撮り直して差し替えてください。

設定ファイルについて

設定ファイルは以下にまとめられています：

src/fishbot/config/


通常利用では変更不要。動作調整が必要な場合のみ編集してください。
